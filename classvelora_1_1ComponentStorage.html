Detailed Description

template<typename Component>
class velora::ComponentStorage< Component >

Storage for components of a specific type.

This class manages the storage and retrieval of components associated with entities in the ECS (Entity-Component-System) architecture.

Template Parameters
ComponentThe type of the component to store.
