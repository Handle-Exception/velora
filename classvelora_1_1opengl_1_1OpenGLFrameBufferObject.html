Detailed Description

OpenGL Frame Buffer Object.

This class is used to create and manage an OpenGL Frame Buffer Object (FBO). It provides methods to enable, disable, and check the status of the FBO.

Note
This class is not thread-safe. Must be used in renderer thread.
